## :memo: Descripción del Proyecto

Este proyecto es parte del desafío de **Oracle Next Education (ONE)** en colaboración con **Alura Latam**. El objetivo principal es desarrollar un Conversor de Monedas utilizando el lenguaje **Java**, que realice solicitudes a una API de tasas de cambio (ExchangeRate-API) para realizar conversiones precisas y en tiempo real entre diferentes divisas.

La aplicación permite al usuario interactuar a través de la consola, eligiendo entre diferentes pares de monedas predefinidos o personalizados.

## :hammer: Características de la Aplicación

El conversor ofrece las siguientes funcionalidades:

- `Consumo de API`: Conexión con **ExchangeRate-API** para obtener datos actualizados.
- `Conversión Dinámica`: Cálculo automático de la conversión basado en la entrada del usuario.
- `Menú Interactivo`: Interfaz de consola fácil de usar para seleccionar las divisas.
- `Manejo de Errores`: Gestión de excepciones en caso de entradas inválidas o fallos en la API.
- `Historial (Opcional)`: (Si agregaste historial, menciona esto aquí: Registro de las últimas conversiones realizadas).
